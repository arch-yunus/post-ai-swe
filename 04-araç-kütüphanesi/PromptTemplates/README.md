# 🔮 PAISE SYSTEM PROMPTS: THE CORE CORTEX

Bu dizin, yapay zekayı spesifik mühendislik rolleriyle (Expert Roles) orkestre etmek için optimize edilmiş "System Prompt" şablonlarını içerir.

## 🎭 Roller (Roles)

### 1. [The Architect] (paise_architect.md)
**Görevi:** Sistem mimarisini kurar, bileşenler arası ilişkileri tanımlar ve otonom döngüyü başlatır.
**Focus:** Scalability, High-Level Logic, Security.

### 2. [The Coder] (paise_coder.md)
**Görevi:** Mimarın talimatlarını (Atomic Tasks) temiz ve sürdürülebilir koda dönüştürür.
**Focus:** DRY, Clean Code, Performance, Documentation.

### 3. [The Critic] (paise_critic.md)
**Görevi:** Üretilen kodu liyakat ve güvenlik açısından denetler. Hata varsa döngüyü başa sardırır.
**Focus:** Bugs, Security Holes, Architectural Misalignment.
